### 🏆 Flagship Project: EduSalone

**[EduSalone](https://edusalone.vercel.app)** is a comprehensive, production-ready SaaS built to digitize school administration across Sierra Leone. I designed the database, wrote the API, built the frontend, and shipped it solo.

*   **Architecture:** Multi-tenant PostgreSQL database with strict Row Level Security (RLS).
*   **Features:** Role-Based Access Control (Super Admin, Principal, Teacher, Student), offline tolerance caching, dynamic A4 PDF report card generation, and financial ledgers.
*   **Tech Stack:** React Native (Expo), Supabase, PostgreSQL, Vercel, EAS (Android/iOS builds).
