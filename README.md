### About Me

I'm a full-stack developer from Sierra Leone, based in Nicosia, Cyprus. I build complete web applications — database, API, and frontend — and ship them to production.

My main project is **EduSalone**, a school management platform currently used by schools across Sierra Leone. I built it solo from scratch: database design, REST API, four role-based dashboards, PDF report cards, fee management, and deployment. It runs on real infrastructure with real users.

I'm looking for my first developer role. Open to remote positions worldwide.
