### Flagship Project — EduSalone

A full school management platform built for Sierra Leone schools. Four separate user roles, each with their own dashboard. Schools use it daily to manage students, attendance, results, and fees.

- Role-based access for Admins, Principals, Teachers, and Students
- Student enrolment, attendance register, and exam results
- PDF report card generation
- Fee tracking in Sierra Leonean Leones
- Lesson notes and assignment uploads
- Multi-school SaaS with subscription pricing

**Tech:** React, Node.js, Express, PostgreSQL, Prisma, JWT, Tailwind CSS, Cloudinary, Railway, Vercel
